## Lucide

While Skeleton does not bundle an official icon library, we highly recommend the use of [Lucide](https://lucide.dev/). This provides a huge selection of icons that are available to all popular frameworks, while featuring a clean and modern aesthetic. All code examples within this Skeleton documentation site have been implement using **Lucide**, but feel free to replace with any alternative of your choice.

## Installation

<Framework id="svelte">
	Follow the official instructions to install [Lucide for Svelte](https://lucide.dev/guide/packages/lucide-svelte).
</Framework>
<Framework id="react">
	Follow the official instructions to install [Lucide for React](https://lucide.dev/guide/packages/lucide-react).
</Framework>

## Usage

<Framework id="svelte">
```svelte
<script>
	import SkullIcon from '@lucide/svelte/icons/skull';
</script>

<SkullIcon stroke="pink" class="size-8" />
```
</Framework>
<Framework id="react">
```tsx
import { SkullIcon } from 'lucide-react';

export default function App() {
	return <SkullIcon stroke="pink" className="size-8" />;
}
```
</Framework>

## Sizes

Use `icon-*` to apply standardized icon sizes. The `base` size matches Lucide's default size of `24px`.

<Preview files={{ 'app.astro': SizeRaw }} client:visible>
	<Size />
</Preview>

As well as `icon-label-*` sizes that compliment [badge](/docs/tailwind-components/badges), [button](/docs/tailwind-components/buttons#sizes), and [field](/docs/tailwind-components/forms#field-sizes) sizing.

<Preview files={{ 'app.astro': LabelSizeRaw }} client:visible>
	<LabelSize />
</Preview>

Both utilize a simple convention, making it easy to expand with custom sizes:

```css
@utility icon-4xl {
	--icon-size: 256px;
	width: var(--icon-size);
	height: var(--icon-size);
}
```

```css
@utility icon-label-2xl {
	--icon-size: 24px;
	width: var(--icon-size);
	height: var(--icon-size);
}
```
