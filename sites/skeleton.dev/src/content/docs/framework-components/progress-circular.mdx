## Format

Use the `format` prop to customize the output of the `ValueText` component. Options include:

- `percentage` (default)
- `decimal` (0.0 - 1.0)
- Custom - via the [Intl API](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl).
