## Group

Wrap with `disclosure-group` and share a [`name`](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/details#name) attribute so only one item stays open at a time.

<Preview files={{ 'app.astro': GroupRaw }} client:visible>

</Preview>

## Open

Add the `open` attribute to render a disclosure expanded by default.

<Preview files={{ 'app.astro': OpenRaw }} client:visible>
	<Open />
</Preview>

## Animation

The slide animation is a progressive enhancement via `::details-content` and `interpolate-size: allow-keywords`. For unsupported browsers this will fall back to the native instant toggle.
