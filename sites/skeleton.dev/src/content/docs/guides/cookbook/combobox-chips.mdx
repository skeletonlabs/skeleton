## How It Works

Enable `multiple` on the `Combobox` root to allow selecting more than one item. Each time the user picks an option, `onValueChange` fires with the full list of selected items. Store that list in local state and render each item as a removable `chip` above the input. Clicking ✕ on a chip filters it out of the selected list, which also updates the controlled `value` passed back to the `Combobox`.
