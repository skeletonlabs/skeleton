## Primitives

The basic building blocks that Skeleton typography is built upon.

### Typographic Scale

Primitive font sizes based on your theme's [typographic scale](https://designcode.io/typographic-scales), via the `--type-scale-factor` [theme property](/docs/get-started/plugin#type-scale).

### Semantic Typography

Given that typography can differ wildly from project to project, we've provided an opinionated boilerplate for generating a set of semantic typography. These classes are constructed using a combination of Skelton and Tailwind primitives. Tap the code view, copy the styles to your global stylesheet, then modify as desired. Use the `type-{foo}` prefix to help keep things standardized.
