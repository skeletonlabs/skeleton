## Handling Total Count

If your source data is pre-truncated (ex: server-side pagination). Make sure to specify the total item `count`.

```json
// Mock Server Response
pagination: {
	"page": 1,
	"limit": 2,
	"pages": 1,
	"total": 1, // <----
	"next": null,
	"prev": null
}
```

```tsx
<Pagination count={pagination.total} ... />
```
