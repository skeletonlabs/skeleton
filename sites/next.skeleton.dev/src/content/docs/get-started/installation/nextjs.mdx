## Installation

<Process>
    <ProcessStep step="1">
        ### Create a Project
        Start by creating a new [Next.js project](https://nextjs.org/docs/app/getting-started/installation) via their CLI:
		{/*
        ```console
        npx create-next-app@latest my-skeleton-app --typescript --tailwind --eslint
        cd my-skeleton-app
        ```
		*/}
		{/* prettier-ignore */}
		<div class="card preset-outlined-warning-500 p-8 space-y-8">
			<p>The Next.js CLI has not yet updated to implement <u>Tailwind v4</u> by default, so we recommend following the [official Tailwind guide](https://tailwindcss.com/docs/installation/framework-guides/nextjs) for now. Avoid installing Tailwind during the Next.js CLI prompts.</p>
		</div>
    </ProcessStep>
    <ProcessStep step="2">
        ### Install Skeleton
        Install the Skeleton core and Svelte component packages.
        ```console
        npm i -D @skeletonlabs/skeleton@next @skeletonlabs/skeleton-react@next
        ```
    </ProcessStep>
    <ProcessStep step="4">
        ### Configure Tailwind
        Open your global stylesheet in `/src/app/globals.css` and make the following changes:
        ```css title="globals.css" {3-5}
@import 'tailwindcss';

@import '@skeletonlabs/skeleton';
@import '@skeletonlabs/skeleton/optional/presets';
@import '@skeletonlabs/skeleton/themes/cerberus';
``		Please consult the [Presets](/docs/design/presets) documentation to learn more about this optional feature.
    </ProcessStep>
    <ProcessStep step="5">
        ### Set Active Theme
        Open `/src/app/layout.tsx`, then set the `data-theme` attribute on the HTML tag to define the active theme.
       ``html title="layout.tsx"
<html data-theme="cerberus">...</html>
`
    </ProcessStep>
    <ProcessStep step="check">
        ### Done
        Start the dev server using the following command.
    	`console
npm run dev
```
</ProcessStep>

</Process>
